
My Book Viz:
An interactive web application designed to transform complex literary datasets into intuitive, actionable visualisations.

Overview:
My Book Viz is a React-based platform that allows users to explore and analyse book market trends. By leveraging modern frontend technologies, it provides a responsive interface for viewing data distributions, popularity shifts, and genre-specific insights.

Key Features:
Dynamic Dashboards: Real-time data rendering using React functional components and hooks.

Trend Analysis: Visualises historical and current book data to identify market shifts.

Responsive Design: Fully optimised for desktop and tablet viewing using CSS3 Flexbox and Grid.

Data Integrity: Utilises ESLint configuration to maintain high code quality, consistent formatting, and to catch plug-in errors during development.

Tech Stack
Frontend: React.js

Styling: CSS3

Linting: ESLint (Standard React Template)

Data Source: Kaggle (Detailed below)

Tools: Git, NPM, VS Code

Data Source:
The insights generated by this application are powered by a comprehensive dataset sourced from Kaggle.

Dataset Link: [https://www.kaggle.com/datasets/bahramjannesarr/goodreads-book-datasets-10m/data](https://www.kaggle.com/code/japandata509/eda-amazon-top-50-bestselling-books/notebook)


Processing: The raw data was cleaned and parsed using JavaScript to ensure compatibility with React's state management.

Getting Started
Prerequisites
Node.js (v18.0.0 or higher)

NPM

Installation & Local Development
Clone the repository:

Bash
git clone https://github.com/yourusername/my-book-viz.git
Navigate to the directory:

Bash
cd my-book-viz
Install dependencies:

Bash
npm install
Start the development server:

Bash
npm start
The app will open automatically at http://localhost:3000. ESLint will run in the background to ensure code standards are met.
